# Configuration & Shortcuts

This document describes keyboard shortcuts and configuration options.

## Keyboard Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Cmd/Ctrl + S` | Save workflow | Anywhere |
| `Cmd/Ctrl + N` | Create new workflow | Anywhere |
| `Escape` | Deselect node | Canvas |
| `Tab` | Navigate between focusable elements | Canvas |
| Arrow Keys | Move selection | Canvas |

## Configuration

### Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `LOBSTER_UI_WORKFLOW_DIRS` | Comma-separated workflow directories | - |

### Settings Panel

Access settings via the gear icon in the header.

**Configurable Options**:
- Workflow directories (comma-separated)
- UI theme preference
